📦 Project Name:
iPhone 17 Series Keyboard Case for EXKnight Mini Keyboard

🧠 About this Project:
This project is a custom-designed 3D printable iPhone keyboard case made for the EXKnight Mini Bluetooth Keyboard.
It brings back the classic physical typing experience — inspired by BlackBerry, adapted for modern smartphones.

There are 4 versions optimized for different iPhone 17 models.
The EXKnight keyboard fits snugly into the case while keeping cameras, ports, and buttons accessible.

⚙️ File Information:
- STL files → Ready for 3D printing
- STP (STEP) files → Provided for remixers and CAD users who want to modify or adapt the structure

If you only want to print and use the case → download the STL files.
If you plan to modify or create your own version → use the STP file.
